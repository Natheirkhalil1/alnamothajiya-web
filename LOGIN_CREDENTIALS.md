# بيانات تسجيل الدخول

## حساب المسؤول الافتراضي

**البريد الإلكتروني:** `admin@namothajia.com`  
**كلمة المرور:** `admin123`

## تغيير بيانات الدخول

لتغيير بيانات تسجيل الدخول، قم بتعديل ملف `lib/auth.ts`:

\`\`\`typescript
const ADMIN_USER: User = {
  email: "your-email@example.com",
  password: "your-password",
}
\`\`\`

## ملاحظات الأمان

- هذا النظام مناسب للتطوير والاستخدام المحلي
- لا تستخدم كلمات مرور بسيطة في بيئة الإنتاج
- يُنصح باستخدام Firebase Authentication للإنتاج لمزيد من الأمان
- Firebase Auth يدعم:
  - مصادقة البريد الإلكتروني وكلمة المرور
  - المصادقة الثنائية (2FA)
  - إعادة تعيين كلمة المرور
  - تسجيل الدخول عبر Google وغيرها
